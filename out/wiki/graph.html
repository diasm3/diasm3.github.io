백트래킹 (N-Queen)

백트래킹은 해결책에 대한 후보를 구축해 나아가다 가능성이 없다고 판단되는 즉시 후볼를 포기(백트랙)해 정답을 찾아가는 범용적인 알고리즘으로 제약 충족 문제에 특히 유용하다.

leetcode 문제 https://leetcode.com/problems/n-queens/

1def nqueen(n):
2    visited = [-1] * n # 배열을 n개 만든다 
3    cnt=0 # 카운터 변수 선언
4
5    # 범위를 벗어 났는지 확인
6    def is_ok_on(nth_row):
7        #
8        for row in range(nth_row):
9            if visited[nth_row] == visited[row] or nth_row - row == abs(visited[nth_row] - visited[row]):
10                return False
11        return True
12
13    def dfs(row):
14        if row >= n:
15            nonlocal cnt
16            cnt += 1
17            grid = [['.'] * n for _ in range(n)]
18            for idx, value in enumerate(visited):
19                grid[idx][value] = 'Q'
20            return
21
22        for col in range(n):
23            visited[row] = col
24            if is_ok_on(row):
25                dfs(row + 1)
26
27    dfs(0)
28    return cnt 
29
30print(nqueen(int(input())))

assert nqueen(4) == [[".Q..", "...Q", "Q...", "..Q."], ["..Q.", "Q...", "...Q", ".Q.."]]
