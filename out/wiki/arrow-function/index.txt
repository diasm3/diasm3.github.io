# 5장 Arrow Function
* 기본적인 빌딩 블럭
* 여러번 사용가능
* 작업을 수행 또는 값을 계산

## Function declartion 
- function name(param1, param2){ body... return}
    * 하나의 함수에서는 하나만 해라
    * 이름은 동작하는 작업의 이름으로
    * 함수는 객체이다
    * e.g. createCardnadPoint -> createCard, createPoint
