# 미니프로젝트
* 첫번째 프로젝트로 굉장히 민망하다.

나도 너무나 부족했고 다른분들도 굉장히 힘들었던 프로젝트로 생각했다.
사용했던 스택은 Python, Flask, 그리고 API였다.
가장 힘들었던건 아무것도 모르고 시작하는 Python 사용법이다. 일단 마지막에는 조금 어떻게 돌아가는지 내용을 알았으나 프로젝트를 끝내지 못했다.
