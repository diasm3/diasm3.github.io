# 기본문법

* 문법 시작과 끝을 알려준다
- `\begin
\end`

- 그냥 `$$` 열고 닫아도 된다.
- 곱셈 : `times`
- 덧셈 : `+`
- 아래에 넣기: `_n`
- 제곱 : `^n`

> `$$A + B = A$$` -> 
$$ A + B = A $$

그럼 책에 있는 내용을 넣어볼까?

$$ (N \times ((M \times ( W + T)) + (F \times (1 - W)))) + B $$
